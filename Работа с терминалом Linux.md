### **Навигация и просмотр файлов**

| Команда         | Что делает                                                | Пример                    |
| --------------- | --------------------------------------------------------- | ------------------------- |
| `pwd`           | Показывает **текущую папку** (где ты сейчас)              | `pwd`                     |
| `ls`            | **Показывает список** файлов и папок в текущей директории | `ls`                      |
| `ls -la`        | Показать список ВСЕХ файлов (включая скрытые) подробно    | `ls -la`                  |
| `cd <путь>`     | **Перейти** в другую папку                                | `cd /home/user/Documents` |
| `cd ..`         | Перейти **на уровень выше** (в родительскую папку)        | `cd ..`                   |
| `cd` или `cd ~` | Перейти в **домашнюю директорию** (/home/твой_логин)      | `cd`                      |
| `cd -`          | Вернуться в **предыдущую папку**                          | `cd -`                    |

### **Работа с файлами и папками**

| Команда                                           | Что делает                                                   | Пример                          |
| ------------------------------------------------- | ------------------------------------------------------------ | ------------------------------- |
| `mkdir <имя>`                                     | **Создать папку**                                            | `mkdir Projects`                |
| `touch <имя>`                                     | **Создать пустой файл**                                      | `touch file.txt`                |
| `cat <файл>`                                      | **Показать содержимое** файла на экране                      | `cat file.txt`                  |
| `nano <файл>`                                     | **Открыть файл для редактирования** в простом редакторе Nano | `nano file.txt`                 |
| _(В Nano: Ctrl+X для выхода, Y/N для сохранения)_ |                                                              |                                 |
| `cp <откуда> <куда>`                              | **Скопировать** файл или папку (для папок нужен `-r`)        | `cp file.txt file_backup.txt`   |
| `cp -r <папка> <куда>`                            | Скопировать **папку рекурсивно** (со всем содержимым)        | `cp -r old_folder/ new_folder/` |
| `mv <откуда> <куда>`                              | **Переместить** или **переименовать** файл/папку             | `mv old_name.txt new_name.txt`  |
| `rm <файл>`                                       | **Удалить файл** (навсегда! В корзину не попадает)           | `rm trash.txt`                  |
| `rm -r <папка>`                                   | **Удалить папку** и всё её содержимое                        | `rm -r old_folder/`             |
| `rm -i <файл>`                                    | Удалить с **подтверждением** (спросит 'y/n')                 | `rm -i important.txt`           |

> ⚠️ **Внимание с `rm`!** Удалённое командой `rm` нельзя восстановить простым способом.

### **Просмотр и поиск**

|Команда|Что делает|Пример|
|---|---|---|
|`head <файл>`|Показать **первые 10 строк** файла|`head log.txt`|
|`tail <файл>`|Показать **последние 10 строк** файла|`tail log.txt`|
|`find <где> -name <имя>`|**Найти** файлы по имени|`find . -name "*.txt"`|
|`grep <текст> <файл>`|**Найти строки**, содержащие текст, в файле|`grep "error" log.txt`|

### **Права доступа и информация**

|Команда|Что делает|Пример|
|---|---|---|
|`chmod +x <файл>`|Сделать файл **исполняемым** (чтобы его можно было запустить)|`chmod +x script.sh`|
|`ps aux`|Показать **все запущенные процессы**|`ps aux \| grep chrome`|
|`kill <PID>`|**Завершить процесс** по его ID (увидишь в `ps aux`)|`kill 12345`|

### **Основные трюки и комбинации**

1. **Автодополнение:** Набираешь начало имени файла/папки и жмёшь `Tab`. Система сама предложит варианты или допишет.
    
2. **История команд:** Жми стрелку `Вверх` для прокрутки предыдущих команд. `history` покажет полный список.
    
3. **Очистка экрана:** `clear` или `Ctrl+L`.
    
4. **Прервать выполнение команды:** `Ctrl+C`.
    
5. **Передать вывод команды в файл:**
    
    - `ls > list.txt` — записать вывод `ls` в файл list.txt (перезаписать).
        
    - `ls >> list.txt` — **добавить** вывод `ls` в конец файла list.txt.

### **Короткий пример рабочего процесса:**

```
pwd                     # Смотрю, где я (/home/user)
mkdir test_project      # Создаю папку для проекта
cd test_project         # Захожу в неё
touch README.md         # Создаю файл
nano README.md          # Редактирую его, пишу "Hello World", сохраняюсь (Ctrl+X, Y)
cat README.md           # Смотрю, что получилось
cp README.md README_copy.md # Создаю копию
ls -la                  # Вижу оба файла
cd ..                   # Выхожу из папки
rm -r test_project/     # Удаляю папку проекта (когда она не нужна)
```
